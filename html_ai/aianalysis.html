Analyzing your child's RMM history is quick, easy, and very insightful using Artificial Intelligence. This is because RMM provides a very comprehensive data file for external analysis. A great strength of current AI analyzing data files like RMM's.

Preparing for AI Analysis

To prepare for using AI you need to download two text files from this site, and generate a usage data file using RMM. You likely will want them in an easily accessed folder

This RMM_AI_initiation_instructions.txt file has the instructions you provide initially to the the AI prompt to prepare it for the analysis. Note that each initiation paragraph is a separate prompt entry. If you are using version one of RMM you will want to also enter the third and fourth paragraphs.

The second file is a set of instructions you will upload to the AI prompt prior to uploading the CSV file. This RMM_AI_upload_instructions.txt file has the instructions you will load into the AI prompt. This file was setup for the demo below. You will want to use it as a template to customize the analysis for your child.

Once you have these two files, open RMM and click into the Settings menu (gear icon). The select Stats and your user name. Click Export to CSV File and RMM will create the usage data CSV file. This is automatically stored in the folder your browser is currently using for downloads.

Performing AI Analysis

This AI process was setup to use Google Gemini. To upload files with Gemini you will need to be using a browser window linked to a Google account. This process should also work with ChatGPT, but was not optimized for it.

To use Gemini, open the Gemini web page. If you have previous analysis you may want to delete it. Use the two prompts from the ...initiation... file. Then upload the ...upload_instructions... file. From this point you can have Gemini respond to any question simply by entering "Perform QNN" at the prompt where NN = your desired question number.

Sample AI Analysis

Please note that the CSV data file used for this analysis was quickly created by the software author. Because of this it results will be different from a regular child. Regular use will generate a lot more days of problems for each problem type. Also remember to edit the upload instructions file to customize it for your child.
