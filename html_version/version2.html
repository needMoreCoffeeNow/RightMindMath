RightMindMath Version 2 has one important new feature, and some bug fixes. When you are using the new version a small "v2" text will appear to the right of the Settings (gear) button. If you are happy with the first version, you may choose not to upgrade. Note though that apart from Apple iOS devices, upgrading is quite simple. Even for iOS, it is not that hard. How to upgrade is covered below.

Feature and Fixes

Feature 1: RMM now supports the export and load of its entire database. This allows you to easily change to install RMM on new device and load the database from the old device. Export is quite simple. You just choose the export function from the settings mneu on the old device. This will create a text file. Transfer this text file to the new device, and load it when you first start RMM on the new device.

Fix 1: There was a problem with carries from the 10s column to the 100s column for three digit problems. The carry explanation and the right answer are both affected. If you plan to use RMM for these types of problems you should upgrade.

Fix 2: For two-digit multiplication for one specific problem two records rather than one were posted to the usage database. This overstates the record of problems completed by about 18%. If you are using version one and AI to analyze usage data this can be ignored as you can instruct AI to ignore the problem.

Fix 3: When exporting usage to a .csv file, the final two columns may not have been properly delimited. As with the previous problem, if you use AI to analyze usage you can ignore this. Unless you are exporting the .csv and using Python or Exel to analyze it, you can ignore this bug.

How to Upgrade RMM on a PC and iOS Device

For Computers: Start RMM. On Windows press Ctrl+Shift+I. On Mac press Option+Cmd+I. This will open the Developer Tools Application window. Ensure that "Application" is the active tab at the top of the window. In the left sidebar right-click on the "Cache Storage" and select the "Refresh" option. Close and restart RMM. This will result in the version two files to be downloaded and used when running RMM.

For an iOS Device: Because Apple will not allow you to refresh the RMM cache on the iOS device, you must first connect the device to a Mac. If you do not have a Mac, you can borrow one from a friend as this is a one-time-only process that is safe and secure. First you must enable developer settings on the Mac. Open Safari --> Settings (or Preferences) --> Select the Advanced tab --> Check the "Show features for web developers". Next attach your device to the Mac using a cable. Follow the dialog to allow the Mac to attach to the device. Start RMM on the device. When it opens, go to the Mac and select the iOS device using the "" menu. Once the device is available to Safari on the Mac, follow the instruction above.
